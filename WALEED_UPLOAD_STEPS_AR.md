# خطوات وليد لرفع مشروع Customer Churn Prediction

## 1) أنشئ Repository في GitHub
الاسم:

```text
customer-churn-prediction
```

الوصف القصير:

```text
Machine learning project for customer churn prediction using EDA, feature engineering, model comparison, and Logistic Regression tuning.
```

## 2) ارفع محتويات هذا المجلد كما هي
لا ترفع ملف ZIP نفسه. فك الضغط وارفع الملفات التي بداخله.

## 3) لو تستخدم الأوامر

```bash
cd customer-churn-prediction-ready
git init
git add .
git commit -m "Initial customer churn prediction project"
git branch -M main
git remote add origin https://github.com/weelyX/customer-churn-prediction.git
git push -u origin main
```
